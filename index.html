CHART API LIST

    D3.js

    특징

    1. 많은 데이터를 표현하기 적합하다
    2. 기본 표현을 함수단위로 제공한다.
    3. 많은 양의 그래프를 표현할 수 있다.
    4. HTML의 SVG, CSS를 표현할 수 있다.
    5. 커스터마이징이 자유롭다.

    단점

    1. 구버전과 최신버전의 괴리성
    2. 그래프 표현시, 추가 작업

    Chart.js

    특징

    1. 지속적인 업데이트
    2. 각 구성요소들에 대한 옵션을 제공
    3. 디자이너, 개발자가 사용하기 간단하다.
    4. 반응형 라이브러리
    5. MIT라이선스

    단점

    1. SVG 기반 라이브러리 대비 성능이 좋지 않음.

    Flotr2

    특징

    1. Js프레임워크에 종속적이지 않음
    2. 각 구성요소들에 대한 옵션을 제공
    3. 원본 수정없이 차트 추가 가능
    4. 인쇄시 정확하게 표현되는 차트
    5. MIT라이선스

    단점

    1. 마지막 업데이트가 2014년 7월 22일.
    2. IE8 기술지원은 Flash를 사용

    C3.js

    특징

    1. 지속적인 업데이트.
    2. 커스터마이징이 가능함.
    3. D3를 베이스로 만들어진 라이브버리다.
    4. 반응형 라이브러리
    5. MIT 라이센스

    단점

    1. D3.js에 의존적인 라이브러리다.
